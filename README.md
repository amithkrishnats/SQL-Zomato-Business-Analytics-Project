# Zomato Restaurant Performance & Customer Insights Dashboard

## Project Overview

This project is an end-to-end Business Analytics and Business Intelligence solution developed using SQL, MySQL, Power BI and Excel. The objective of the project is to analyze restaurant performance, customer preferences, pricing patterns, delivery trends and market opportunities using Zomato restaurant data.

The project demonstrates practical applications of SQL for data cleaning, exploratory analysis and business problem-solving, while Power BI is used to create an interactive dashboard that provides actionable insights for business decision-making.

## Tools & Technologies

* SQL
* MySQL
* Power BI
* Microsoft Excel
* GitHub

## Key Business Questions

* Which cities have the highest restaurant concentration?
* Which cuisines are most preferred by customers?
* How does pricing impact restaurant ratings?
* Does online delivery influence customer satisfaction?
* Does table booking improve customer ratings?
* Which cities offer potential expansion opportunities based on ratings and competition levels?

## Key KPIs

* Total Restaurants
* Total Cities
* Average Rating
* Total Votes
* Restaurants with Online Delivery
* Restaurants with Table Booking

## Dashboard Features

* City-wise Restaurant Distribution
* Cuisine Analysis
* Rating Distribution
* Online Delivery Analysis
* Table Booking Analysis
* Price Range vs Rating Analysis

## Business Insights

* Identified high-potential cities for restaurant expansion.
* Evaluated the impact of online delivery on customer ratings.
* Analyzed customer cuisine preferences across locations.
* Assessed pricing strategies and their relationship with restaurant performance.
* Generated actionable insights for business growth and customer engagement.
